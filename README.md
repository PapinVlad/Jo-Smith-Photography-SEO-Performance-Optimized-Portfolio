This is a high-performance, SEO-centric landing page for a professional photographer based in Dundee. Unlike my previous projects, the main challenge here wasn't just "making it work," but making it rank high on search engines and provide a flawless user experience on any device or connection speed.


Structured Data (JSON-LD): I implemented Schema.org rich snippets (ProfessionalService). This tells Google exactly what the business is, its location (Dundee), and price range, which helps the site appear as a "rich result" in search.

Metadata Masterclass: I didn't just add a title; I implemented Open Graph (OG) and Twitter Card tags to ensure that when the site is shared on social media, it looks professional with the correct image and description.

Local SEO: I added specific Geo-tags (geo.position, geo.region) and a canonical URL to help the site dominate local search results in Scotland.

Performance Optimization: * I used WebP image formats and the loading="lazy" attribute for all non-hero images to keep the page size tiny.

I added preconnect links to external CDNs to reduce the Time to First Byte (TTFB).

Accessibility (A11y): I used semantic HTML (<main>, <nav>, <figure>) and ARIA roles to make sure the site is fully navigable for users with screen readers.
